# Code of Conduct

This project aims to be straightforward, respectful, and useful to contributors and users.

## Expected Behavior

- Be respectful and professional in discussions and reviews.
- Focus feedback on the code, behavior, and design rather than the person.
- Assume good intent, but be precise when pointing out problems.
- Help keep the project welcoming to contributors with different backgrounds and experience levels.

## Unacceptable Behavior

- Harassment, abuse, threats, or discriminatory language
- Personal attacks or deliberately hostile review behavior
- Publishing private information without permission
- Repeated bad-faith disruption of issues, pull requests, or discussions

## Maintainer Responsibility

Project maintainers may edit, remove, or reject contributions and discussions that do not meet these standards.

## Reporting

If you run into a serious conduct issue, open a private channel with the maintainer rather than escalating publicly.
